你好，我是 Creased

一个热爱设计与技术的创作者，相信好的文字值得好的排版。

关于这里

Creased，意为「折痕」。每一次思考都像是一次折叠，在纸面上留下痕迹——有时清晰，有时模糊，但都是真实的记录。

这个博客创建于 2025 年秋天。最初只是想有一个安静的角落记录技术笔记，后来逐渐延伸到设计感悟、阅读心得和日常观察。我希望这里的每一篇文章都经过认真的思考和打磨，而不是流水账式的产出。

关于我

一个前端工程师，同时对 UI/UX 设计、排版和东方美学有浓厚兴趣。工作之外，喜欢阅读、散步、喝咖啡，偶尔写写随笔。

我相信技术和美学不是对立的——好的代码应该像好的文章一样清晰优雅，好的设计应该像好的建筑一样兼具功能与美感。

技能与兴趣

TypeScript React Vue Astro CSS / Design System Node.js UI/UX 设计 排版 Figma 摄影 阅读 咖啡

关于这个博客

本站使用 Astro 构建，追求简洁与性能的平衡。设计灵感来自东方美学中的留白理念和 Apple 官网的极简排版。字体采用 Noto Serif SC 和 Inter 搭配，兼顾中西文排版美感。
